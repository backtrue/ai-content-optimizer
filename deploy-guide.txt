========================================
📝 接下來的步驟
========================================

步驟 1️⃣：推送到 GitHub
------------------------

1. 前往 GitHub 創建新倉庫：
   https://github.com/new

2. 倉庫設定：
   - 名稱：ai-content-optimizer
   - 描述：AI 內容優化大師
   - 可見性：Public 或 Private
   - ⚠️ 不要勾選 "Initialize with README"

3. 創建後，在終端執行（替換 YOUR_USERNAME）：

   git remote add origin https://github.com/YOUR_USERNAME/ai-content-optimizer.git
   git push -u origin main


步驟 2️⃣：部署到 Cloudflare Pages
----------------------------------

1. 登入 Cloudflare Dashboard：
   https://dash.cloudflare.com/

2. 選擇 "Workers & Pages" → "Create application" → "Pages"

3. 點擊 "Connect to Git" → 選擇 GitHub

4. 選擇 ai-content-optimizer 倉庫

5. 構建設定：
   ┌─────────────────────────────────────┐
   │ Framework preset: None              │
   │ Build command: npm run build        │
   │ Build output: dist                  │
   └─────────────────────────────────────┘

6. 環境變數（選擇一個）：
   
   選項 A - OpenAI（推薦）：
   OPENAI_API_KEY = sk-your-key-here
   
   選項 B - Gemini（免費）：
   GEMINI_API_KEY = AIza-your-key-here
   
   選項 C - 暫時跳過（使用 Mock 數據）

7. 點擊 "Save and Deploy"

8. 等待 2-5 分鐘完成部署


步驟 3️⃣：獲取 API Key（如果還沒有）
------------------------------------

OpenAI：https://platform.openai.com/api-keys
Gemini：https://makersuite.google.com/app/apikey
