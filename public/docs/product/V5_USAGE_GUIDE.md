# v5 自適應混合評分 - 使用指南

## 快速開始

### 1. 提交內容分析

#### 步驟 1：進入分析頁面
訪問 AI 內容優化大師的分析頁面。

#### 步驟 2：貼上內容
- 支援純文字或 HTML 格式
- 建議 300 字以上
- 系統會自動偵測格式

#### 步驟 3：輸入關鍵字
- 最多 5 個關鍵字
- 用逗號分隔
- 例如：「葉黃素, 眼睛健康, 補充品」

#### 步驟 4：輸入 Email
- 分析完成後將寄送結果連結
- 請確保 Email 正確

#### 步驟 5：提交
點擊「提交分析」按鈕，您的內容將進入分析隊列。

### 2. 等待結果

#### 預計時間
- 分析通常需要 **1-5 分鐘**
- 高峰期可能需要 10-15 分鐘

#### 查看進度
- 系統會顯示任務 ID
- 您可以保存此 ID 以備查詢

#### 接收 Email
- 分析完成後，您將收到一封 Email
- Email 中包含完整結果連結
- 點擊連結即可查看詳細分析

## 理解評分

### 雙軌評分系統

v5 模型使用兩個獨立的評分維度：

#### 結構分（40% 權重）
衡量內容的**組織與表現**

**對於 HTML 內容**
- 標題結構（H1/H2 層級）
- 段落、清單、表格
- 可讀性（句子長度）
- 引用與數據

**對於純文字內容**
- 內容長度
- 段落結構
- 可讀性
- 引用與數據

**改進建議**
- 增加清單或表格
- 拆分長段落
- 補充引用與數據

#### 策略分（60% 權重）
衡量內容的**說服力與完整性**

**WHY - 問題定義（1-10 分）**
- 是否清楚描繪讀者痛點？
- 是否交代「為什麼讀者需要這篇文章」？

**HOW - 實現方法（1-10 分）**
- 是否解釋解決方案的原理？
- 是否提供具體步驟或指引？

**WHAT - 解決方案（1-10 分）**
- 是否清楚總結解決方案？
- 是否強調對讀者的實際幫助？

**改進建議**
- 加強問題描述的具體性
- 補充更多實務案例
- 明確總結解決方案

### 總分解讀

```
總分 = 結構分 × 0.4 + 策略分 × 0.6
```

**分數範圍**
- **80-100**：🌟 優秀 - 內容品質卓越，已具備 AI 引用潛力
- **60-79**：👍 良好 - 內容有基礎，可進一步優化
- **40-59**：⚠️ 中等 - 需要改進，建議按建議調整
- **0-39**：❌ 需改進 - 建議重新調整內容策略

## 常見問題

### Q: 為什麼我的純文字評分比 HTML 低？
**A:** 純文字模式無法檢查 HTML 標籤（如 H1/H2）。建議：
- 補充更多內容（1000+ 字）
- 改進段落結構
- 補充引用與數據

### Q: WHY 分數很低，該怎麼辦？
**A:** 這表示您的文章缺乏問題描述。建議：
- 在開篇明確指出讀者面臨的痛點
- 用具體例子說明問題的重要性
- 交代「為什麼讀者需要這篇文章」

### Q: HOW 分數低意味著什麼？
**A:** 您的內容缺乏具體的執行指引。建議：
- 補充分步驟的說明
- 加入實務案例或操作指南
- 解釋解決方案的原理

### Q: WHAT 分數低怎麼辦？
**A:** 您的結論或解決方案表述不清。建議：
- 在結尾明確總結解決方案
- 強調解決方案對讀者的實際幫助
- 避免模糊或過於通用的結論

### Q: 多久會收到 Email？
**A:** 通常 1-5 分鐘。如果超過 15 分鐘未收到：
- 檢查垃圾郵件資料夾
- 確認 Email 地址正確
- 使用任務 ID 手動查詢結果

### Q: 可以重新分析同一篇文章嗎？
**A:** 可以。每次提交會生成新的任務 ID 和分析結果。

### Q: 結果會保存多久？
**A:** 結果在 KV 中保存 7 天。建議保存 Email 或截圖以備日後查閱。

## 最佳實踐

### 撰寫高分內容

#### 1. 清楚的問題定義（提升 WHY）
```
❌ 不好：「本文介紹葉黃素補充品。」
✅ 好：「隨著年齡增長，許多人面臨視力衰退的困擾。葉黃素是一種關鍵營養素，可幫助保護眼睛。本文將指導您如何選擇最適合的補充品。」
```

#### 2. 具體的執行步驟（提升 HOW）
```
❌ 不好：「選擇一個好的補充品。」
✅ 好：「選擇補充品時，應考慮以下因素：
1. 劑量：建議 10-20mg 每日
2. 形式：游離形式比酯化形式吸收更快
3. 認證：尋找 USP 或 NSF 認證
4. 搭配：與玉米黃質結合效果更佳」
```

#### 3. 明確的結論（提升 WHAT）
```
❌ 不好：「選擇合適的補充品很重要。」
✅ 好：「總結來說，選擇葉黃素補充品需要考慮劑量、形式和認證。建議從 10-20mg 的劑量開始，選擇知名品牌，並持續 3-6 個月以觀察效果。」
```

#### 4. 補充證據（提升結構分）
- 引用研究數據：「2021 年的研究發現...」
- 提供具體數字：「改善視力對比度敏感度 35%」
- 列舉案例：「許多消費者報告...」

#### 5. 改進可讀性（提升結構分）
- 使用清單和表格
- 拆分長段落（< 380 字）
- 簡化句子結構（平均 15-25 字）

- **純文字**：直接貼上文本

### 內容限制
